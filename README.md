# Login Validator (ログイン情報検証ツール)

依頼人（クライアント）から受け取った Web サイトのログイン情報が「正しいか」「作業に必要な権限があるか」を一瞬で確認し、整理するための検証ツールです。

## 主な機能
1.  **デュアル入力検証**: 
    - 「WordPress」と「サーバー管理画面（Xserver等）」の 2 組の情報をセットで入力可能。
    - 一括クリックで両方の検証を同時に実行し、結果を分かりやすく表示。
2.  **WordPress 管理者権限チェック**: 
    - 単なるログイン成功だけでなく、ログイン後のダッシュボード画面を解析し、プラグイン操作などが可能な「管理者 (Administrator) 権限」があるかを自動判定。
3.  **WAF・セキュリティ回避**: 
    - **トップページ経由アクセス**: サーバーの WAF（攻撃検知）を回避するため、まずトップページを訪問してセッションを確立してからログイン画面へ向かう「人間らしい挙動」を再現（HTTP 403 エラーへの対策）。
    - **独自URLへの対応**: SiteGuard 等でログインURL（`wp-login.php`）が変更されていても、ページの「中身」で WP かを見抜き、カスタムパスを尊重して検証を実行。
4.  **コピペ用レポート生成**: 
    - 検証結果をご指定の「＜作業に必要な情報＞」形式で自動生成。
    - パスワードを伏せずに表示・コピー可能で、開発者（ユーザー）が迷わず正しい情報を手元に控えられます。

## 技術スタック
- **Frontend**: Next.js (App Router), Tailwind CSS
- **Backend**: Node.js (API Route)
- **Library**: 標準 http/https モジュール (SSL証明書の警告を回避しつつ接続チェックを行うため)

## 開発の目的
このツールは、開発者と依頼人の間での「ログインできない」「パスワードが違う」「権限が足りない」といった不毛なやり取りをゼロにすることを目的としています。
依頼人に本ツールで事前確認をしてもらうことで、誰が送ってきた情報か、権限は十分か、を瞬時に裏取りできます。
