# web-login-validator

Webログイン認証情報のバリデーション・接続確認ツールです。

クラウドソーシング等で受けたWordPress・レンタルサーバー等の作業依頼において、クライアントから受け取ったログイン情報（ログインURL・ユーザID・パスワード）が正しいかどうかを素早く検証します。

---

## 背景・目的

作業依頼の際、クライアントから送られてくるログイン情報は以下の問題を抱えていることが多い：

- 必要な情報が揃っていない（URLなし、IDなし等）
- フォーマットが統一されていない
- 情報は揃っているが実際にはログインできない

これらの問題を自動で検出し、クライアントへの再送依頼メッセージを自動生成することで、情報収集にかかる往復コミュニケーションを削減します。

---

## 機能概要

1. **フォーマットバリデーション** — ログインURL・ユーザID・パスワードの形式チェック
2. **実接続チェック** — 実際にログインを試みて成否を確認
3. **エラー原因の特定** — URL不達 / ID誤り / PW誤り / その他
4. **再送依頼メッセージ生成** — NGの場合、クライアントへ送る文面を自動生成

---

## 対応ログイン種別

| 種別 | 例 |
|------|----|
| WordPress 管理画面 | `https://example.com/wp-admin` |
| レンタルサーバー サーバーパネル | Xserver, ConoHa, さくら等 |
| その他 Webフォームログイン | 管理画面全般 |

---

## 入力情報

| 項目 | 必須 | 説明 |
|------|------|------|
| ログインURL | ✅ | ログインページのURL |
| ユーザID | ✅ | ログインに使用するID |
| パスワード | ✅ | ログインに使用するパスワード |
